
### mason 




#### 🎯 Activate from https://pub.dev
dart pub global activate mason_cli

Alternatively, mason_cli can be installed via homebrew

#### 🍺 Install from https://brew.sh

brew tap felangel/mason
brew install mason

#### 🚀 Initialize mason
mason init
